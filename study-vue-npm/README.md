
# vue hello-world#

## 环境前提 ##
安装vue-cli

    npm install --global vue-cli

## 创建应用 ##

    vue init webpack my-app

## 启动应用 ##

    vue run dev 

## 打包应用 ##

    vue run build 

## 问题处理 ##
index页面打开空白

config 下index.js
assetsPublicPath：'/' 替换为 assetsPublicPath：'./'
